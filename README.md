# Quiz JS App

A simple JavaScript quiz application that can be run locally and customized easily.

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone <your-repo-url>
cd <your-repo-name>
```

2. Run the demo:

Open the file **`quizz_demo`** located at the root of the project in your browser.

---

## 🎮 Controls

The quiz is fully navigable using the keyboard:

* **↑ / ↓ (Arrow Up / Arrow Down)** → Navigate between answer choices
* **Enter** → Validate your selected answer

---

## ✏️ Customize the Quiz

You can easily modify or create your own quiz:

### Option 1: Edit the existing demo

* Open the demo file
* Modify the questions directly

### Option 2: Create your own quiz

1. Copy the demo file
2. Update the question data (text, answers, etc.)
3. Change the data source if needed

---

## 📁 Structure

* `quizz_demo` → Main demo file to launch the quiz
* Question data is defined directly in the demo or linked source files

---

## 💡 Notes

* No build step required — just open the file in a browser
* Designed for quick testing and easy customization
